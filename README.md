# 🌍 Eco Swap

A simple and interactive web app that helps users replace everyday harmful items with eco-friendly alternatives.

---

## 📌 About the Project

Eco Swap is built to promote sustainable living by helping users make small but impactful changes in their daily lives.

Users can search for common items (like plastic bottles or plastic bags) and get:
- eco-friendly alternatives
- environmental impact
- practical tips

---

## ✨ Features

- 🔍 Search-based eco suggestions  
- 🔄 Visual swap animation (❌ → ↓ → ✅)  
- 🌍 Environmental impact awareness  
- 🌱 “Why this matters” explanation  
- 💡 Actionable eco-friendly tips  
- 🎨 Clean and minimal UI  

---

## 🛠 Tech Stack

- React (Vite)
- Tailwind CSS
- JavaScript
